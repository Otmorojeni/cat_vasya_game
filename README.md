# cat_vasya_game


## Описание работы ##


В данной практической работе, помогаем коту Василию сделать свою игру. Идея игры "Игра от кота Василия" заключается в создании ролевой игры с пошаговой боевкой. Где есть персонажи и монстры. У каждого есть характеристики: сила, ловкость, телосложение, мудрость, интеллект, харизма. Кот Василий уже набросал арзитектуру на бумажке, но он не может печать на клавиатуре и мы ему в этом поможешь. Наша задача помочь коту Василию реализовать классы на Python и выполнить каждый модуль в соответсвии с заданием.


### Модуль 1 ###


В данном модуле было реализовано следующее - создан абстрактный класс Unit, где в дальнейшем от него наследуются классы Character и Monster. В конструктор класса Unit передаются 6 характеристик о юните. Далее были созданы абстрактные методы - calculate_max_health(), calculate_damage(), calculate_defense(). После, были созданы дочерние классы - Character и Monster. В них была реализована логика работы абстрактных методов в соответсвии с формулами кота Василия.


### Модуль 2 ###


В этом модуле была переработана архитектура игры. Убран класс Monster и переработан класс Character. Теперь класс Character не абстрактный, он реализовывает методы но с ветвлением логики в зависимости от нового атрибута - character_class, который может включать только - warrior, mage, hunter. В зависимости от класса персонажа будут вычислятся его параметры по предоставленным формулам кота Василия. В конструкторе сразу происходит вычисление параметров персонажа и проверка на корректность класса персонажа.
